# How to use the text component?
Most of the time, the QA and design team finds minor text issues that are related to `font-size`, `line-height`, and `font-weight`. So, part of standardising the **text component** is kind of important in every design system. This guide helps you to avoid text related issues. Follow on..!

You can use this search box to search for font-size/weight or line-height to get the component. Just copy the code and use your text.
